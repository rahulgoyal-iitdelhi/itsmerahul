Intermodal Hub Location for Indian Railways

This project focuses on optimizing the placement of intermodal hubs in the Indian railway freight network using mathematical modeling and optimization techniques.

Highlights:

  • Formulated an MILP (Mixed Integer Linear Program) to model hub selection and flow routing.
  • Minimized total cost including fixed hub setup and transportation costs.
  • Used Python, Gurobi, and Pyomo for model development and solving.
  • Improved route efficiency and load balancing among major terminals.
