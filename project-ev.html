Optimal Placement of EV Charging Network

This project aims to design a cost-effective and accessible EV charging infrastructure through data-driven optimization.

Highlights:

  • Clustered demand points using K-means for scalable location planning.
  • Applied MILP with waiting time constraints and maximum entropy principle.
  • Balanced between fast and slow charging stations with dynamic cost modeling.
  • Used Python, SCIP, and Gurobi solvers for implementation.
